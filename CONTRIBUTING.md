# Contributing to EvolvSQL

Thanks for your interest in EvolvSQL! We're building a PostgreSQL-compatible database from the ground up using BEAM (Erlang/Elixir) and Rust.

## Current Status

**We are not accepting contributions at this time.**

The project is in its earliest stages and the architecture is still being defined. We need to establish the core foundation before we can effectively coordinate external contributions.

## When Will Contributions Open?

We'll open contributions once:

- The core architecture is stable
- The Rust storage engine handles basic CRUD
- We have a test suite and CI pipeline
- Contribution guidelines and code standards are documented

Watch this repo or check back — we'll update this file when we're ready.

## In the Meantime

- Star the repo if you find the idea interesting
- Open issues for feature requests or discussion
- Read `FEATURES.md` for the full scope of what we're building
